Nhom : Ten gi cung duoc
1 byte -> 128

Package (Sent):
	4 byte(128) : message length
	4 byte(128) : senderID
	4 byte(128) : destinationID
	1 byte : type of message(Group, Upload, Download, Message) 
	(data)

Package (Recv):
	4 byte(128) : message length
	4 byte(128) : senderID
	4 byte(128) : groupID
	1 byte : type of message(Group, Upload, Download, Message) 
	(data)



Single: S recvId Message
Group : G groupID message
Upload : S recvID UPLOAD (fileName)
		 G recvID UPLOAD (fileName)
Download: S recvID DOWNLOAD (fileID)
		  G groupID DOWNLOAD (fileID)

Cmd : G 0 CREATE (id)
	  G destID LEAVE
	  S uid LIST
	  G gid LIST

Upload/Download: 8 byte(128) fileSize + data
